# 3D Digital Twin Asynchronous AI API Server

Educational project for building an asynchronous AI API server
for 3D digital twin surveillance systems.

## Tech Stack
- FastAPI
- Redis / Celery
- YOLOv8, BLIP-2
- Docker

## Structure
- api/        : FastAPI server
- worker/     : Celery workers
- datasets/   : Test datasets (git submodule)
